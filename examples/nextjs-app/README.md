# Vulyo Next.js example

A minimal App Router application showing Vulyo sign-in, sign-up, waitlist, account management, route protection, and the first-party auth proxy.

```bash
cp .env.example .env.local
pnpm install
pnpm dev
```

Create an application in the Vulyo dashboard, then place its publishable and secret keys in `.env.local`. The example runs at `http://localhost:3001`.
