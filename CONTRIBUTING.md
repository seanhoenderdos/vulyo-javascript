# Contributing

Issues and focused pull requests are welcome.

1. Open an issue for behavioral or public API changes.
2. Create a branch from `main`.
3. Run `pnpm install --frozen-lockfile` and `pnpm verify`.
4. Add a Changeset when a published package changes.
5. Keep pull requests scoped and include tests for changed behavior.

Never commit credentials, customer data, private service code, or generated build output.
